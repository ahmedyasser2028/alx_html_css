# CSS Advanced Project

## Objective
This project is focused on implementing advanced CSS styling for a webpage following a Figma design. The goal is to practice CSS layout, colors, fonts, and responsive design.

## Files
- `index.html`: Main HTML file for the page
- `styles.css`: CSS file containing all the styles

## Instructions
1. Open `index.html` in a browser.
2. Make sure `styles.css` is linked in the `<head>` tag.
3. All sections (Header, Banner, Quotes, Videos list, Membership, FAQ, Footer) follow the Figma design.
4. Use the Figma file to match colors, fonts, and spacing.
5. Fonts used:
   - [Source Sans Pro](https://fonts.google.com/specimen/Source+Sans+Pro)
   - Spin-Cycle-OT (download from Figma if needed)
