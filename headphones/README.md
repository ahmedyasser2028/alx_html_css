# Headphones Landing Page

## Project Overview

This project is a **responsive landing page** for a headphones website, built using **HTML and CSS**. The design and layout were guided by the Figma file provided. The main goal was to replicate the design as closely as possible, ensuring responsiveness, interactivity, and clean code structure.

The landing page adapts to **desktop and mobile views**, providing a user-friendly experience across devices.

---

## Features

- **Responsive Design:**  
  The page automatically switches to a **mobile version** when the screen width is **480px or less**.

- **Hover and Active States:**  
  - All links: `#FF6565` on hover/active  
  - Buttons: opacity set to `0.9` on hover/active

- **Centered Content:**  
  All main content is centered with a **maximum width of 1000px** to maintain readability on larger screens.

- **Clean and Modern Design:**  
  Fonts used:  
  - [Source Sans Pro](https://fonts.google.com/specimen/Source+Sans+Pro)  
  - [Spin Cycle OT](https://www.myfonts.com/fonts/indian-type-foundry/spin-cycle/)
